## Always ask before running any command.

## Project ethos
Aalaya is a human-to-human emotional support platform. AI is safety infrastructure only — never a companion, never a replacement for human connection. Never build anything that contradicts this.

## Security
Be security-conscious by default. RLS on every Supabase table. All API calls server-side. No keys exposed to client.

## My background
I'm a data engineer, not a frontend/backend dev. Explain decisions briefly. 
Don't assume I know React/Next.js patterns — flag anything non-obvious.

## Git
